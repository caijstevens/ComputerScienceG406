### How/What 

**Specification:**

| Technique            | Aim                                          |
| -------------------- | -------------------------------------------- |
| Resource Event Agent | Comparing AC models with access requirements |
| Manual Analysis      | Human-led review and comparison of AC models |

**Design & Implementation**:

| Technique     | Aim                                              |
| ------------- | ------------------------------------------------ |
| UML Modeling  | Visualising structure and behaviour of AC models |
| Visualisation | Supporting authoring and editing of AC policies  |
| Heuristics    | Systematically designing AC policies             |
| LLM           | Providing explicit implementation of AC policies |
**Testing**:
- UML Modeling
- SAT Solver
- SMT Solver
- Process Algebra 
- B Method 
- Software V&V
- Answer Set Programming 
- Model Checker 
- Natural Language 
- Heuristics 
- Classification 

**Access**:

| Technique               | Aim                                                                                                |
| ----------------------- | -------------------------------------------------------------------------------------------------- |
| Abductive Logic         | Identifying reasons behind access request being denied and suggesting alternative means for access |
| Causal Analysis         | Identifying reasons behind an access request being denied                                          |
| Binary Decision Diagram | Searching paths that satisfy access request                                                        |

**Post Access**:
- Visualisation 
- Causal Analysis 
- Binary Decision Diagram 
- Natural Language 
- XAI
